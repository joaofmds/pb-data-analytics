				- 2 – Liberdade de distribuir cópias do software para ajudar ao próximo;
				- 3 – Liberdade de melhorar o software e liberar/disponibilizar as modificações, de modo que toda a comunidade se beneficie

		- Histórico
			- 1965 - MULTICS
			- 1969 - MULTICS reescrito com o Assembly
			- 1973 – UNIX reescritona linguagem C
			- 1976 – A AT&T opta por “fechar” o código do UNIX 
			- padrão POSIX
			- 1983 – Richard Stallman inicia o Projeto GNU
			- 1987 – Andrew S. Tanenbaum desenvolve o SO MINIX
			- 1991 – Linus Torvalds desenvolve um Kernel a partir do MINIX
			- 1991 – Linus disponibiliza a primeira versão oficial do Kernel Linux
		- Movimento / Projeto GNU
